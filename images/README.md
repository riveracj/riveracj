# Images Folder

Place your images here for the portfolio.

## Required Images

### Profile Photo
- **File name:** `profile.jpg`
- **Recommended size:** 400x400px
- **Format:** JPG or PNG
- **Max file size:** 200KB
- **Notes:** Professional headshot or photo

### Project Screenshots
- **File names:** `project1.jpg` through `project6.jpg` (or however many you have)
- **Recommended size:** 600x400px
- **Format:** JPG or PNG
- **Max file size:** 300KB each
- **Notes:** Clear screenshots or mockups of your projects

## Image Optimization

Before uploading, optimize your images:

1. **Use online tools:**
   - [TinyPNG](https://tinypng.com) - Free compression
   - [Squoosh](https://squoosh.app) - Google's image optimizer
   - [CompressJPEG](https://compressjpeg.com) - Bulk compression

2. **Recommended tools:**
   - Photoshop: "Save for Web"
   - GIMP: Export with quality 85%
   - Online tools above

3. **Target sizes:**
   - Profile photo: < 200KB
   - Project images: < 300KB each
   - Total all images: < 2MB

## Image Guidelines

### Profile Photo
- Use good lighting
- Professional attire (or casual if that's your brand)
- Clear background (or blurred)
- Smile and look approachable
- Centered and well-framed

### Project Screenshots
- Full project view OR
- Most important feature highlighted
- Clean, professional look
- Avoid text-heavy screenshots
- Show the UI/UX clearly

## Alternative: Use Placeholder Services

If you don't have images yet, the portfolio uses placeholders automatically:
- Profile: `https://via.placeholder.com/400x400?text=Your+Photo`
- Projects: `https://via.placeholder.com/600x400?text=Project+1`

These will display until you add real images.

## Supported Formats

- JPG/JPEG (recommended for photos)
- PNG (recommended for logos/graphics with transparency)
- WebP (modern format, best compression)
- SVG (for icons and logos)

## File Naming

Use simple, lowercase names:
- ✅ `profile.jpg`
- ✅ `project1.jpg`
- ❌ `My Profile Photo.JPG`
- ❌ `Project #1 Screenshot.jpeg`

---

**Pro Tip:** Keep the original high-resolution images in a separate folder as backup!
