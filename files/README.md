# Files Folder

Place downloadable files here, such as your resume/CV.

## Required Files

### Resume/CV
- **File name:** `resume.pdf`
- **Format:** PDF (recommended)
- **Max file size:** 2MB
- **Notes:** Keep it updated!

## Resume Tips

### Creating Your Resume

1. **Keep it concise:** 1-2 pages maximum
2. **Use professional template**
3. **Include:**
   - Contact information
   - Professional summary
   - Work experience
   - Education
   - Skills
   - Projects (optional)
   - Certifications (if relevant)

### Free Resume Tools

- **Canva:** [canva.com](https://canva.com) - Great templates
- **Google Docs:** Free templates
- **Microsoft Word:** Built-in templates
- **Overleaf:** [overleaf.com](https://overleaf.com) - LaTeX resumes
- **Resume.io:** [resume.io](https://resume.io) - Online builder

### Converting to PDF

**From Word/Google Docs:**
- File → Download → PDF
- File → Save As → PDF

**Online Converters:**
- [smallpdf.com](https://smallpdf.com)
- [ilovepdf.com](https://ilovepdf.com)

### Optimization

- Keep file size under 2MB
- Use readable fonts (10-12pt)
- Ensure it's ATS-friendly (if applying to jobs)
- No password protection
- Test on different PDF readers
